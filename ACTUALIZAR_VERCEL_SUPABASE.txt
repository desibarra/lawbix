## PASO 2: Actualizar Variables en Vercel

Ejecuta estos comandos UNO POR UNO:

```powershell
cd c:\Users\desib\Documents\lawbix\backend

# Remover variables MySQL antiguas
vercel env rm DB_HOST production

vercel env rm DB_PASSWORD production
vercel env rm DB_NAME production
vercel env rm DB_PORT production

# Agregar variables Supabase (reemplaza con tus valores)
echo "aws-0-us-east-1.pooler.supabase.com" | vercel env add DB_HOST production
echo "postgres.xxxxx" | vercel env add DB_USER production

echo "postgres" | vercel env add DB_NAME production

```
